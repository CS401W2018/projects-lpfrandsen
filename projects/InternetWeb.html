Internet

A global network of computers and servers communicating and exchanging data.

Web

Interlinked hypertext documents accesed using the Interent

Difference between Internet and Web

The Internet is what links computers together and the Web is a service that runs on the internet

ASCII

A character encoding system. Uses 7 bits to represent 128 characters

Unicode

Another character encoding standard that can represent millions of characters

UTF-8

Encoding for Unicode that ensures compatibility with ASCII

URL

The address used to access something inside the Web

Domain Name

A readable address that identifies a specific location on the Internet

HTTP

Protocol that allows web browsers and servers to communicate

HTTPS

A security extension of HTTP that secures the data transfer

IP Address

Unique numerical label assigned to each device on a network

Domain Name System

A system used to translate readbale names into IP addresses that computers can identify

W3C Recommendations

Guidelines from the World Wide Web Consortium to ensure a long term future and growth

XML

A language made for the transportation of data which can be easily read by humans and computers

XHTML

A type of HTML that enforces stricter rules to improve conssistency of web documents

HTML

The standard language for creating web pages. Defines the structure of web documents

HTML5

The fifth revision of HTML which introduces new attributes to support modern web applications
