What are special entity characters

Special characters within html that are not found on a traditional keyboard. Each special entity character has a number and a name and either one can be used to print the character desired.They all start with a & and end with a ;.

What is the difference between OL and UL Tags

Ordered Lists (OL), will sort lists into a predescribed orderd. The order in these lists does matter. Unordered Lists (UL), will not sort the items on a list. There is no ranking system or other way of numbering the list.
